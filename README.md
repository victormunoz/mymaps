# Mymaps

## Technology

- Node.js version 20 or higher.
- Mongo version 8 or higher.
- Render for deployment

## Local Installation and Run

```
   cd Server
   npm i
   npm run start
```

```
   cd Client
   npm i
   npm run execute
```

## Database

Mongo

String connection in .env (MONGO_URI)

## Publish

Push to master branch.
